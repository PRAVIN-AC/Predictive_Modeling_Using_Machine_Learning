## 🤖 Task 2: Predictive Modeling Using Machine Learning

An ensemble **Random Forest Classifier** was trained to predict penguin species based on structural morphometric features (`bill_length_mm`, `bill_depth_mm`, `flipper_length_mm`, and `body_mass_g`).

### 📈 Evaluation Summary
* **Model Selected:** Random Forest Classifier ($100$ estimators)
* **Dataset Split:** 80% Training / 20% Testing (Stratified)
* **Overall Test Accuracy:** 98.55%
